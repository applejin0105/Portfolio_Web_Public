# Jongjin Portfolio

인터랙티브 개인 포트폴리오 웹사이트. (React SPA)
🔗 Live: https://jongjinportfolio.com

## Tech Stack
- **Frontend**: React, Vite, React Router, Framer Motion, tsParticles, Howler
- **Backend**: Node.js, Express (간단한 인증 API)
- **Dev**: Storybook

## Features
- 페이지 전환 애니메이션 / 사운드 인터랙션
- 비밀번호 인증 후 접근하는 문서 뷰어
- 반응형 스케일링 레이아웃

## Run Locally
```bash
cd frontend/react
npm install
npm run dev
